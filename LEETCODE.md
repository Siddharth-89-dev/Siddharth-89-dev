## 💡 Problem-Solving Approach

### Strategy
1. **Understand the Problem** — Read carefully and identify constraints
2. **Plan the Approach** — Decide on the algorithm and data structure
3. **Code** — Implement with clean, readable code
4. **Test** — Verify with edge cases
5. **Optimize** — Improve time and space complexity if needed

### Techniques I Use
- **Arrays & Strings** — Sliding window, two-pointer, prefix sums
- **Dynamic Programming** — Top-down, bottom-up, memoization
- **Graphs & Trees** — BFS, DFS, binary search trees
- **Hashmaps & Sets** — Frequency counting, lookup optimization
- **Sorting & Searching** — Binary search, merge sort, quick sort
